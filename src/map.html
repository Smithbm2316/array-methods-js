Map

Use map() when you want to take an array of pre-existing data and manipulate that data, without affecting the original array. 

          
            const names = [
              "Ben Smith", 
              "John Michaelson", 
              "Wes Tomkins", 
              "Julie Donaldson", 
              "Katelyn Roberts"
            ];
            const firstNames = names.map(name => name.split(" ")[0]);
            // returns ["Ben", "John", "Wes", "Julie", "Katelyn"];
