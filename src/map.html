What is map()?

As per the Mozilla Developer Network (MDN) web documenation, "The map() method creates a new array populated with the results of calling a provided function on every element in the calling array." Unlike forEach(), map() returns an array with the same length as the array it was called on. map() allows you to take an array of pre-existing data and manipulate that data, without affecting the original array. 

            
              const names = [
              "Ben Smith", 
              "John Michaelson",

              const firstNames = names.map(name => name.split(" ")[0]);
              // returns ["Ben", "John", "Wes", "Julie", "Katelyn"];

This has a plethora of uses in Javascript. In the snippet above, we are given an array called names where we have a list of 5 names, including each person's first and last name in the string. If I wanted an array of those names, but only with the first names included, I can use map() to easily get that data without changing the original names array.

We define a variable firstNames that is set equal to the value that names.map() returns. We pass a parameter of name which will be set equal to each of the strings in names, and map() will return the first name by calling name.split(" ")[0], which splits up each string into an array of values, splitting those strings whenever there is a space character. So, in the example of "Ben Smith", name.split(" ") would be equal to ["Ben", "Smith"], and the [0] grabs the first position in that array and returns that value ("Ben").
