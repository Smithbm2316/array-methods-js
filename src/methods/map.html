What is map()?

As per the Mozilla Developer Network (MDN) web documentation, "The map() method creates a new array populated with the results of calling a provided function on every element in the calling array." Unlike forEach(), map() returns an array with the same length as the array it was called on. map() allows you to take an array of pre-existing data and manipulate that data, without affecting the original array. It takes in a callback function as an argument, which only has one required parameter: currentValue. The currentValue will be equal to the element that we are currently iterating over within map(). There are several other optional parameters (such as index, which is quite useful) that are not within the scope of this post, but you can read about on the Mozilla Developer Network (MDN) web documentation. map() has a plethora of uses in Javascript. In the snippet below, we are given an array called names where we have a list of 5 names, including each person's first and last name in the string. If I wanted an array of those names, but only with the first names included, I can use map() to easily get that data without changing the original names array. 

            
              const names = [
                "Ben Smith", 
                "John Michaelson", 
                "Wes Tomkins", 
                "Julie Donaldson", 
                "Katelyn Roberts"
              ];
              const firstNames = names.map(name => name.split(" ")[0]);
              // returns ["Ben", "John", "Wes", "Julie", "Katelyn"];

We define a variable firstNames that is set equal to the value that names.map() returns. We pass an argument of name to our callback function, which will be set equal to each of the strings in names. map() will return the first name by calling name.split(" ")[0], which splits up each string into an array of values, splitting those strings whenever there is a space character. So, in the example of "Ben Smith", name.split(" ") would be equal to ["Ben", "Smith"], and the [0] grabs the first position in that array and returns that value "Ben". 

            
              const people = [
                { firstName: "Ben", lastName: "Smith", age: 21 },
                { firstName: "John", lastName: "Michaelson", age: 43 },
                { firstName: "Wes", lastName: "Tomkins", age: 32 },
                { firstName: "Julie", lastName: "Donaldson", age: 67 },
                { firstName: "Katelyn", lastName: "Roberts", age: 27 },
              ];

              const ages = people.map(person => person.age);
              // returns [21, 43, 32, 67, 27]

In our example above, we define an array of objects called people, and we want to get an array of all of the ages of the people in our array. map() makes it super simple! We pass person as our only argument to our callback function that we provide to map(), and return person.age for each person.
