What is filter()?

As per the Mozilla Developer Network (MDN) web documentation, "The filter() method creates a new array with all elements that pass the test implemented by the provided function.” Like map(), filter() allows you to take an array and return a new array, without manipulating the original array that you call filter() on. Also like map(), filter() takes in a callback function as an argument, which only has one required parameter: element. The element will be equal to the element that we are currently iterating over within filter(). There are several other optional parameters (such as index, which is quite useful) that are not within the scope of this post, but you can read about on the Mozilla Developer Network (MDN) web documentation. Let’s jump into some examples! 

            
              const names = [
                "Ben Smith",
                "John Michaelson",
                "Wes Tomkins",
                "Julie Donaldson",
                "Katelyn Roberts",
              ];

              const firstLetterIsJ = names.filter(name => name[0] === "J");
              // returns [“John Michaelson”, “Julie Donaldson”];

In the above example, we have an array called names. However, we want a new array that only contains names that start with the letter “J”. filter() makes this a trivial task! We simply pass name to our callback function, and in our callback we return whether or not the first character in name is equal to “J”. If our callback function returns true, then filter() adds that item (in this case name) to a new array and moves onto the next iteration. If our callback returns false, then filter() ignores that item and moves onto the next iteration. Finally, after filter() has iterated through our entire array, it returns a new array with our names that start with “J”. Easy enough! 

            
              const people = [
                { firstName: "Ben", lastName: "Smith", age: 21 },
                { firstName: "John", lastName: "Michaelson", age: 43 },
                { firstName: "Wes", lastName: "Tomkins", age: 32 },
                { firstName: "Julie", lastName: "Donaldson", age: 67 },
                { firstName: "Katelyn", lastName: "Roberts", age: 27 },
              ];
                          
              const underThirty = people.filter(person => person.age < 30);
              /* 
              returns [
                { firstName: "Ben", lastName: "Smith", age: 21 },
                { firstName: "Katelyn", lastName: "Roberts", age: 27 },
              ];
              */

What if we wanted to get all of the people in an array of people objects that have an age property that is less than thirty? filter() also make this extremely easy! In the above example, we call filter() on our people array, and pass person as our only argument to the callback that we provide to filter(). In our callback we return whether that person’s age (in this case, person.age property) is less than 30. If it is, filter() adds that object to a new array and keeps on iterating. If that person’s age is 30 or older, filter() ignores that object and keeps on iterating. Finally, filter() finishes iterating through our people array and returns a new array to underThirty that contains only the objects that have an age property that is less than 30.

I hope this introduction to filter() was helpful to you! filter() is a very simple way of narrowing down pre-existing data without ever manipulating the original array of data. I encourage you to experiment with it and find new ways to integrate filter() into your projects!
